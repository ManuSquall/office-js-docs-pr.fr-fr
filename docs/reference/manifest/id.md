# <a name="id-element"></a>Élément Id

Spécifie l’ID unique de votre complément Office comme GUID.

**Type de complément :** application de contenu, de volet Office, de messagerie (Mail)

## <a name="syntax"></a>Syntaxe

```XML
<Id>string</Id>
```
